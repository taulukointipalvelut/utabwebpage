Utab

by taulukointipalvelut

Utab is a free tabulation software with high functionality. It supports NA, BP, Asian, Academic(high school), Academic(university) and PDA format.

Currently Utab is developed separately; local part for creating matchups and online part for collecting results of tournament. It takes more time for integrated version.


Volunteer Developers are wanted.

More details

We developed Utab aiming for easier preparation of tabulation, more functions to make best matchups, richer usability to speed up tournaments.

Local part

Local part of Utab can generate matchups based not only on power-pairing but also on other criteria you set. It covers various styles of debate.

Current latest version is ver3.40.

Read more...

Online part

Online part of Utab is a data collecting program set up on online server. You can use this program to show matchups online, collect debaters' or adjudicators' score and export the results. Currently only supporting PDA format. NA supporting version will soon be available.
